# Skills Index

用途：快速定位本机已装 skill 的主用途，避免重复导入和重复决策。

## 文档

- `docx` - Word 文档读写与编辑
- `pdf` - PDF 读取、拆分、合并、填充
- `pptx` - 幻灯片创建、编辑、转换
- `xlsx` - 表格读取、编辑、生成
- `doc-coauthoring` - 结构化长文档协作写作
- `internal-comms` - 内部沟通、公告、FAQ、周报

## 营销与 SEO

- `seo-audit` - 页面或站点级 SEO 技术审计，含脚本化检查与 HTML 报告
- `geo-audit` - GEO / AI 可见性技术审计

## 设计与创意

- `canvas-design` - 静态视觉设计与版式输出
- `algorithmic-art` - 算法生成艺术
- `brand-guidelines` - 品牌规范与视觉一致性
- `slack-gif-creator` - Slack 适配的动图生成

## 系统预装

- `skill-creator` - 创建和优化 skill 的系统工具

## 维护规则

- 每次新增、删除或替换 skill 后，更新这里。
- 每个领域优先只保留一个主 skill。
- 只写导航，不复制各 skill 的详细说明。
